# Template Syntax

## Vue

**Interpolation & Directives:**
Uses double curly braces (`{{ }}`) for binding data.

```html
<template>
  <p>Hello, {{ name }}!</p>
</template>
```

## React

**JSX:**
Uses single curly braces to embed JavaScript expressions.

```jsx
function Greeting({ name, setName }) {
  return (
    <div>
      <p>Hello, {name}!</p>
    </div>
  );
}
```

## Angular

**Template Syntax:**
Uses double curly braces for interpolation.

```html
<p>Hello, {{ name }}!</p>
```
